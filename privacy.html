YesNo Privacy Policy

Last updated and effective: July 3, 2020

This privacy policy applies to the Discord bot YesNo#9598.

When a poll is created with YesNo, the poll text will be stored in the bot's database along with the automatically-generated Discord Channel ID and Message ID (Discord IDs look like 399174767610953728). No other data relating to the server is stored. YesNo never stores data specific to users.

The data is stored only for the bot to function and is not shared with anyone else or used for any other reason. The data is stored on Google Cloud.

To delete the data stored for a channel, send yn.close in it.

To contact me (Advaith), the developer of YesNo, you can mention advaith#9121 (190916650143318016) in the AdvaithBot Discord server.

Also see Discord's Privacy Policy.

Thanks for using YesNo!
